## Requirements

- [GCC](https://gcc.gnu.org/) 13.2+ is required
- [CMake](https://cmake.org/) 3.16+ is required
- [XMake](https://xmake.io/) 2.5+ is required
- [Git](https://git-scm.com/) 2.25+ is required&

- [Boost](https://www.boost.org/)
